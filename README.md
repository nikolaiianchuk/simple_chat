Написал сервер и клиент для чата, общающиеся по TCP друг с другом.

Вообще если отвлечься от TCP, для общения с JS в браузере я бы
выбрал WebSockets, а для обмена данными с другими сервисами и/или
мобильными приложениями я бы выбрал gRPC.

Сервер захардкожен на 50000 порт. Запускается через обычный go run main.go

Клиенту передаются два параметра:
- ConnectionString, например localhost:50000
- Никнейм в чате, например Alice или Bob
